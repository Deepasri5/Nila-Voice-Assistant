# Nila - AI Voice Assistant

## 📌 Overview
Nila is an AI-based Voice Assistant developed using Python.  
It performs voice recognition, automation tasks, system operations, and advanced scientific calculations.

## 🚀 Features
- Speech Recognition using Google API
- Text-to-Speech conversion
- WhatsApp message automation
- Google and Wikipedia search
- Scientific Calculator
- File & Directory Management
- Camera access
- IP Address detection

## 🛠 Technologies Used
- Python
- pyttsx3
- SpeechRecognition
- OpenCV
- pywhatkit
- Wikipedia API
- Requests

## 📄 Publication
Presented and published at an IEEE International Conference.
