# 🗣️ Nila - AI Voice Assistant

## 📌 Project Overview
Nila is an AI-based Voice Assistant developed using Python that performs speech recognition, automation tasks, system control, and intelligent operations using voice commands.

The system listens to user commands, processes them using natural language interpretation, and executes corresponding tasks.

---

## 🎯 Objective
To design and develop an intelligent voice-controlled assistant capable of:
- Understanding voice input
- Performing automation tasks
- Executing system-level operations
- Providing real-time information

---

## 🚀 Key Features
- 🎙 Speech Recognition using Google Speech API
- 🔊 Text-to-Speech response system
- 📩 WhatsApp message automation
- 🌐 Google and Wikipedia search integration
- 🧮 Advanced scientific calculator
- 📁 File and directory management
- 📷 Camera access
- 🌍 IP address detection
- 🖥 Application control (Notepad, Edge, etc.)

---

## 🧠 System Workflow
1. Capture voice input using microphone
2. Convert speech to text
3. Analyze user query
4. Match command logic
5. Execute corresponding system action
6. Provide voice response

---

## 🛠 Technologies Used
- Python
- pyttsx3 (Text-to-Speech)
- SpeechRecognition
- OpenCV
- pywhatkit
- Wikipedia API
- Requests
- OS & System libraries
